## 1. ОСНОВНАЯ МЕХАНИКА: «ПЕРЕМАНИВАНИЕ»

### Как это работает

Каждый игрок управляет своим клубом. Клубы **существуют в одном городе** и конкурируют за гостей.

```
Игрок A (Studio 57)          Игрок B (Club Paradise)
      │                               │
      ▼                               ▼
  Очередь гостей ◄──── ГОРОД ────► Очередь гостей
                           │
                   Гости выбирают клуб
                   на основе репутации,
                   диджея и слухов
```

### Механика переманивания

1. **Репутация влияет на поток:** Клуб с Репутацией 80 получает больше «качественных» гостей автоматически
2. **Активное переманивание:** Можно потратить деньги на «агента», который переманит гостей из конкурирующего клуба
3. **Диверсии:** Слухи, подставы, доносы полиции на конкурента (повышает его Police Heat)

## 2. ТИПЫ МУЛЬТИПЛЕЕРНЫХ РЕЖИМОВ

### Режим А: «Открытый город» (Asynchronous PvP)

- 10–50 игроков в «городе» (Telegram-сервер)
- Каждый играет в своё время (нет синхронизации)
- Рейтинг обновляется каждые 24 часа
- Взаимодействие: репутационные атаки, переманивание

**Идеально для Telegram** — не нужна онлайн-синхронизация.

### Режим Б: «Ночная битва» (Live PvP, будущее)

- 4 игрока одновременно в реальном времени
- Одна общая «ночная смена» на 4 клуба
- Одни и те же гости: кто быстрее одобрит/откажет
- Live лидерборд по прибыли каждую секунду

### Режим В: «Клубный альянс» (Co-op, будущее)

- 2–3 игрока управляют ОДНИМ клубом совместно
- Разделение ролей: Фейсконтроль / Финансы / Охрана
- Общая прибыль, общий FBI риск
- Нужна координация в реальном времени

## 3. МЕХАНИКИ ВЗАИМОДЕЙСТВИЯ

| Действие | Стоимость | Эффект |

| **Слух в прессе** | $500 | Reputation конкурента -10 на 1 ночь |

| Защита | Стоимость | Эффект |
|---|---|---|
| **Пресс-служба** | $400/ночь | Иммунитет к «Слуху» |

## 4. TELEGRAM-СПЕЦИФИЧНАЯ РЕАЛИЗАЦИЯ

### Клубный чат
- При создании клуба → автоматически создаётся Telegram-группа клуба
- Союзники видят события в реальном времени
- «Репортёры» пишут в чат: «Ночь была горячей!»

### Рейдерство через TG
```
[Игрок A] отправляет [Игроку B]:
«Моя звезда в эту ночь — твоя, за $2000»
[Принять] / [Отклонить]
```

### Еженедельный турнир
- Каждые 7 дней: особая «легендарная ночь»
- Все игроки соревнуются за звание «Клуб недели»
- Приз: эксклюзивный скин, Stars, место в галерее Hall of Fame

---

## 5. ANTI-CHEAT В МУЛЬТИПЛЕЕРЕ

```
Все финансовые операции → верифицируются сервером
GameState.velvetBox → только сервер авторизует изменения
Все событийные roll → сид хранится на сервере, не у клиента
Подозрительные паттерны → автоматическая блокировка
```
