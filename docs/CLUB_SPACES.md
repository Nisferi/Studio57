# Пространства клуба — Экстерьер и интерьер
## Velvet & Stash: Studio 57

---

## КОНЦЕПЦИЯ

Клуб — это три мира, которые игрок переключает в зависимости от задачи.  
Каждое пространство — отдельная Phaser сцена (или слой сцены), со своими механиками.

```
[УЛИЦА / ОЧЕРЕДЬ]     ←→    [ФОЙЕ / ФЕЙСКОНТРОЛЬ]
       ↑                              ↓
[ОФИС — УПРАВЛЕНИЕ]           [ВНУТРИ КЛУБА]
                                      ↓
                              [ПОДВАЛ] (версия max)
```

---

## ПРОСТРАНСТВО 1: УЛИЦА И ОЧЕРЕДЬ

### Что это
Экстерьер клуба. Ночной Нью-Йорк. Очередь из 50–200 человек.  
Здесь начинается каждая ночная смена — **до того**, как гости дойдут до твоего поста.

### Что видит игрок
```
[Здание 57-й студии — слева]
[Неоновая вывеска]
[Лимузин у бордюра]
[Толпа — пиксельные силуэты, 3 ряда в глубину]
[Паразит с камерой — справа]
[Арни у боковой двери — иконка для диалога]
[Полицейская машина вдали — если Police Heat > 50]
```

### Механика «Очередь»
До начала фейсконтроля — **20-секундный pre-night phase**:

1. **Управление хайпом**: можно «закрыть» вход на 30 сек → очередь растёт → репутация +5 (искусственный дефицит)
2. **Первичный отбор**: охранник Стэн выталкивает явных неадекватов (пьяных в хлам). Тебе — только сложные случаи.
3. **Паразит**: журналист/папарацци у входа. Разрешить фото (хайп) / запретить (может написать плохо).
4. **Дилер в переулке**: только для версий adult/max. Можно организовать «вход для дилера» через боковую дверь.

### Интерактивные объекты
- **Лимузин**: кнопка «Придержать вход» — задержать очередь пока VIP выходит. +$300 за услугу.
- **Арни у стены**: диалог-советник перед ночью. Даёт подсказку.
- **Мусорный бак в переулке**: для версии max — место где прячут что-то тёмное.

### Визуал
- Пиксельные многоуровневые толпы (silhouette-style, 3 слоя параллакса)
- Мигающая неоновая вывеска
- Такси проезжают в фоне
- Снег зимой / жара летом (зависит от месяца в игре)

## ПРОСТРАНСТВО 2: ФЕЙСКОНТРОЛЬ (уже реализовано)

Текущая NightScene. Игрок стоит у входа, проверяет гостей.  
*Улица* и *Внутренность* видны как фоновые слои.

### Улучшения к текущей сцене
- **Глубина**: за спиной видны силуэты танцующих через стеклянные двери
- **Очередь на фоне**: анимированные силуэты людей ждут
- **VIP-линия**: отдельный вход для знаменитостей (правее основного)

## ПРОСТРАНСТВО 3: ВНУТРИ КЛУБА (новое)

### Что это
Вид изнутри во время ночи. Ты не управляешь напрямую — это **панорамная информационная панель**, которую можно открыть кнопкой прямо во время ночи.

> Это не отдельная сцена, а **слайдер/таб** поверх NightScene: свайп вправо = заглянуть внутрь.

### Что видит игрок внутри

```
┌──────────────────────────────────────────────────────┐
│  [ТАНЦПОЛ]      [БАР]      [VIP]      [СЦЕНА]        │
│                                                       │
│  Пиксельные танцоры    Стойка с бутылками    Пусто /  │
│  (анимация по BPM)     Бармен работает       Концерт  │
│                                                       │
│  Vibe: ████████░░  78%                               │
│  Dance floor: 47 people                              │
│  Bar queue: 12 people                                │
└──────────────────────────────────────────────────────┘
```

### Зоны внутри

#### Танцпол
- Визуальный индикатор «энергии»: плотность танцоров
- Знаменитость на танцполе → особая анимация (свет направлен на неё)
- При FBI > 60% → один из танцоров «не танцует» (агент наблюдает)
- При Police Heat > 70% → мигающие синие огни сквозь окна

#### Бар
- Видно текущую очередь к бару (визуальный аналог bar-тика)
- Апгрейд «Премиум бар» виден в интерьере (меняется стойка)
- Пьяный гость у бара → иконка предупреждения

#### VIP-секция (ложа)
- Видны знаменитости которые сейчас внутри
- Нажать на знаменитость → мини-информация: «Lisa Monelli: bar boost ×2.5 активен»
- Пустая ложа → «Пригласи звезду»

#### Сцена
- По умолчанию: DJ-будка (влияет на бар-доход при Sound апгрейде)
- При забронированном концерте: сцена и оборудование
- Во время концерта: специальная анимация

### Управление из интерьера
Игрок **не может** управлять гостями изнутри — это только мониторинг.  
НО: можно нажать одну кнопку:
- 🚪 «Закрыть бар» — временно останавливает bar-тик, зато снижает Police Heat
- 💡 «Сменить свет» — меняет атмосферу (Vibe + репутация)
- 🎵 «Громче» / «Тише» — влияет на Vibe и риск жалоб соседей

## ПРОСТРАНСТВО 4: ПОДВАЛ (версия max only)

### Что это
Basement — реальная зона Студии 54, куда охрана не совалась.  
Доступен только в версии «Максимум», после разблокировки улучшения «Подвальный вход».

### Механики подвала

**Пассивный доход**: подвал приносит $50–200 в тик без твоего участия (если оборудован)

**Особые события** (только в версии max):
- «VIP-оргия» → можно держать компромат → blackmail mechanism
- «Частная вечеринка дилера» → налаженный поток дохода, но FBI +5/ночь пассивно
- «Журналист пробрался» → кризис (молчание за деньги или скандал)

**Управление**: отдельный охранник для подвала (найм в офисе). Если нет охраны → события выходят из-под контроля.

## ОФИС (уже реализован — улучшения)

Офис остаётся «дневным» пространством между ночами. Дополнения:

### Новые объекты в офисе
- **Стенная карта клуба**: схема всех пространств, видно что куплено/нет
- **Телефон**: звонки от Арни, Коллинза, поставщиков
- **Мусорная корзина**: случайно выброшенный конверт от IRS
- **Сейф в стене**: визуал тайника — чем больше накоплено, тем «тяжелее» выглядит сейф

| Пространство | Разблокируется | Стоимость |

| VIP-ложа улучшенная | VIP Lounge ур.1 | $1500 |

## ПЕРЕХОДЫ МЕЖДУ ПРОСТРАНСТВАМИ

```
МЕНЮ → [ОФИС] (день)
             ↓
       [УЛИЦА] (pre-night, 20 сек)
             ↓
    [ФЕЙСКОНТРОЛЬ] (основной геймплей)
         ←→ [ВНУТРИ КЛУБА] (свайп/таб)
             ↓
       [КОНЕЦ НОЧИ]
             ↓
       [ОФИС] (следующий день)
```

### Анимации переходов
- День → Ночь: затемнение + появление неона
- Снаружи → Внутри: «открыть дверь» — звук толпы, музыка становится громче
- Ночь → Конец: постепенное затухание огней

## ТЕХНИЧЕСКИЕ ЗАМЕТКИ

### Рендеринг внутреннего вида
Внутренний вид = отдельный Phaser Container, скрытый по умолчанию.  
Активируется через `this.interior.setVisible(true)` при свайпе.  
Танцоры: процедурные спрайты (как портреты гостей), анимация через Phaser.Tweens синхронизированная с BPM из AudioSystem.

### Толпа на улице
Силуэты: упрощённые пиксельные фигуры 8×16 пикселей.  
50–200 силуэтов — не рисуются по одному. Используется Phaser.GameObjects.RenderTexture для батчинга.

### Параллакс фона
Три слоя:
- Дальний план: здания (медленный скролл при движении)
- Средний: очередь
- Передний: фонари, такси, детали
