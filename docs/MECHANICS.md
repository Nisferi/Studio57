# Механики игры — Полное описание
## Velvet & Stash: Studio 57

---

## 1. ФЕЙСКОНТРОЛЬ (Papers Please Style)

### Параметры каждого гостя

| Параметр | Значения | Влияние на решение |
|---|---|---|
| **Возраст (шо ID)** | 16–40 лет | Несовершеннолетние → штраф $500 + ФБР +15% |
| **Реальный возраст** | ≠ ID при фальшивке | Требует внимательности |
| **Фальшивый ID** | 65% chance у несовершеннолетних | Не помечен явно — надо замечать |
| **Стиль** | Ультра/Опрятный/Стильный/Обычный/Паршивый | Паршивый → репутация -2 |
| **Состояние** | Трезв/Навеселе/В хлам | В хлам → риск драки |
| **VIP-флаг** | Знаменитость | Бонус при входе, особые эффекты |

### Правила (меняются по ночам)

| Ночь | Минимальный возраст | Особые правила |
|---|---|---|
| 1–3 | 18+ | Только возраст и явные буйные |
| 4–8 | 21+ | Дресс-код обязателен для стиля |
| 9+ | 21+ | Ужесточённая проверка, случайные ловушки |

### Тайминг
- Гость появляется каждые **3.2–5.0 секунды** (рандом)
- Если не принял решение за **6 секунд** → гость уходит сам (штраф к репутации)
- Ночь длится **55 секунд** реального времени

### Последствия решений

**ПРАВИЛЬНЫЕ:**
- Впустил хорошего гостя → +$50–$150 (касса)
- Отказал несовершеннолетнему → 0 (нейтрально)
- Впустил VIP → +$500–$3000 (зависит от знаменитости)

**ОШИБКИ — ВПУСТИТЬ:**
- Несовершеннолетний → 40% chance → полиция → штраф $500 + ФБР +15%
- Пьяный в хлам → 40–50% chance → драка → ущерб $300 + репутация -3
- Паршивый → репутация -2 (немедленно)

**ОШИБКИ — НЕ ВПУСТИТЬ:**
- VIP-гость отказан → репутация -5 + пропущенный бонус

## 2. ФИНАНСОВАЯ СИСТЕМА

### Velvet Box (Официальная касса)
```
Входящий поток:
  + Билеты гостей ($25–$150 зависит от стиля)
  + Бар-тик каждые 5 сек ($25–75 зависит от уровня бара)
  + Бонусы знаменитостей (разовые)

Исходящий поток:
  - Штрафы за нарушения
  - IRS налог 30% (в конце ночи)
  - Покупки улучшений (в офисе)
```

### Stash (Тайник)
```
Входящий поток:
  + «Спрятать» кнопка: -$200 из кассы → +$200 в тайник
  + Некоторые события (взятка дилеру, корруп. сделки)

Риски:
  - Каждое сокрытие: FBI Suspicion +12%
  - Рейд ФБР → ВЕСЬ тайник конфискуется
  - Уровень подозрения ≥15% → шанс рейда = (FBI% / 100) * 0.02 каждую секунду
```

### Налоговая система
```
В конце каждой ночи:
  Чистая прибыль = Velvet Box * (1 - 0.30)
  
  Пример:
  Касса: $2000
  Налог: -$600 (30%)
  Остаток: $1400
  
  Тайник: $800 (нетронут, если нет рейда)
  Итого активов: $2200
```

## 3. FBI И ПОЛИЦИЯ

### Шкала ФБР (FBI Suspicion) 0–100%

| Уровень | Статус | Что происходит |
|---|---|---|
| 0–15% | Зелёная зона | Рейды невозможны |
| 15–40% | Жёлтая зона | Рейды редки (~1% в секунду) |
| 40–70% | Оранжевая зона | Рейды вероятны (~3% в секунду) |
| 70–100% | Красная зона | Рейд почти неизбежен (~5% в секунду) |

**Что повышает FBI Suspicion:**
- Каждый спрятанный мешок: +12%
- Штраф за несовершеннолетних: +15%
- Отказ от взятки полицейскому: +15%
- Игнорирование наркосделки: +10%
- Скандальное событие (VIP-комната): +20%

**Что снижает FBI Suspicion:**
- Дать взятку «Детективу Коллинзу»: -25%
- Улучшение «Охрана» уровень 3: пассивно -5% в ночь
- Провести ночь без нарушений: -3%

### Шкала Полиции (Police Heat) 0–100%

Отдельная шкала. Влияет на:
- Частоту полицейских проверок
- Штрафы за нарушения (Police Heat > 50% → штрафы x2)
- При Police Heat > 80% + FBI > 40% → Game Over

## 4. СЛУЧАЙНЫЕ МАКРО-СОБЫТИЯ

Каждую ночь с определёнными вероятностями происходят события.

### Bianca & The Horse (Бьянка и конь)
- **Шанс:** 12% начиная с Ночи 2
- **Версии:** Soft (символы), Medium (текст), Max (иллюстрация)
- **Выбор:** Разрешить (хайп +40%, риск ущерба $400) / Отказать (ничего)

### Drug Deal (Наркосделка)
- **Шанс:** 15% начиная с Ночи 1
- **Версия Семейная:** «Подозрительная сделка» (💊 без деталей)
- **Версия 18+:** «Кокаин в туалете» (текст)
- **Версия Максимум:** Пиксельная иллюстрация сцены
- **Выбор:** Игнорировать / Выгнать / Дать взятку

### VIP Room Incident (Инцидент в VIP)
- **Шанс:** 10% начиная с Ночи 2
- **Версия Семейная:** «Закрытая вечеринка» 
- **Версия 18+:** «VIP-инцидент» (намёки)
- **Версия Максимум:** Иллюстрация сцены
- **Выбор:** Разрешить (риск скандала или хайп) / Запретить

### Corrupt Cop (Коррумпированный офицер)
- **Шанс:** 13% начиная с Ночи 1
- **Детектив Коллинз** — коррумпированный полицейский, предлагает «закрыть глаза»
- **Выбор:** Дать $500 (FBI -25%, Police Heat -30%) / Отказать (Police Heat +15%)

### Журналист у входа
- **Шанс:** 8% начиная с Ночи 3
- Хочет войти с фотоаппаратом
- **Выбор:** Впустить (репутация +20% или скандальные фото) / Отказать

### Знаменитость требует особых условий
- **Шанс:** 20% когда знаменитость в очереди
- Конкретные требования: определённый диджей, вид напитка, охрана
- Соответствуешь → +500 бонус / Не соответствуешь → VIP уходит

## 5. СИСТЕМА ЗНАМЕНИТОСТЕЙ

### Как они появляются
- Начиная с Ночи 2: 10% шанс что вместо обычного гостя придёт знаменитость
- Знаменитость требует определённого уровня улучшений
- В очереди выглядит как обычный гость, но стиль ULTRA и особая пометка

## 6. РЕПУТАЦИЯ

**Репутация (0–100)** — публичный образ клуба.

**Влияет на:**
- Базовую стоимость билетов (+1% за каждые 10 очков)
- Частоту появления знаменитостей
- Тип гостей (высокая репутация → меньше паршивых)

**Растёт от:**
- VIP-гости в клубе: +5–15
- Апгрейд «Световое шоу»: +5–20 за ночь
- Успешная ночь без инцидентов: +3

**Падает от:**
- Впустить паршивого гостя: -2
- Отказать знаменитости: -5
- Скандал (СМИ, полиция): -10–20

## 8. ПРОГРЕССИЯ СЛОЖНОСТИ

### Ночь 1–3 (Учебный режим)
- Только возраст и явные буйные
- FBI Suspicion стартует с 0
- Мало штрафов
- Нет знаменитостей

### Ночь 4–8 (Опасная зона)
- Дресс-код строже
- Фальшивые ID чаще
- Знаменитости появляются
- FBI начинает расти
- Macro-события активны

### Ночь 9–15 (Золотая / Смертельная)
- Все механики на максимуме
- FBI за одну ночь может вырасти с 0 до 60%+
- Два гостя могут быть «агентами ФБР в прикрытии»
- Финальный выбор: продать клуб / продолжать / бежать

## 9. IDLE-МЕХАНИКА (БУДУЩЕЕ)

Для Telegram: когда игрок офлайн, клуб продолжает работать (Idle режим):
- Пассивный доход ~50% от активного
- Накапливается до 8 часов
- Возвращаешься — собираешь «ночную кассу»
- FBI Suspicion не растёт в idle
- Случайные события не происходят в idle
