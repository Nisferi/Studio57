# Версии контента — Три режима
## Velvet & Stash: Studio 57

---

## Концепция

Игра поставляется в трёх версиях контента, которые игрок выбирает **один раз при первом запуске**. Можно изменить в настройках.

Все три версии — **одна игра**, только с разными слоями нарратива и визуала.

---

## ВЕРСИЯ 1: «Чистая» (Family Edition)

**Возрастной рейтинг:** 12+  
**Платформы:** Telegram (все возрасты), Steam (обычная версия)

### Правила контента
- Наркотики: заменены на «💊 подозрительная сделка» или «🎲 странное поведение»
- Секс: только «💋 закрытая вечеринка» без деталей
- Коррупция: «🤝 смазали руки» — символически
- Насилие: только текстовое «⚡ стычка»
- Эротика: отсутствует полностью

### Визуальные события
- Нет пиксельных сцен с 18+ контентом
- Комикс-кадры: только абстрактные (тени, силуэты, символы)
- Персонажи одеты всегда

### Примеры событий в этой версии
```
💊 «ПОДОЗРИТЕЛЬНАЯ СДЕЛКА»
«В углу происходит что-то странное.
Охрана доложила о необычном обмене.»
[Игнорировать] [Разобраться]

💋 «ЗАКРЫТАЯ ВЕЧЕРИНКА»
«Кто-то устроил частную тусовку в подсобке.
Риск: скандал. Выгода: хайп.»
[Разрешить] [Запретить]
```

## ВЕРСИЯ 2: «18+» (Papers Please Edition)

**Возрастной рейтинг:** 18+  
**Платформы:** Steam (18+ метка), Telegram (с предупреждением)

### Правила контента
- Наркотики: прямое называние (кокаин, таблетки, etc.), текстово
- Секс: намёки и описания без явных изображений
- Коррупция: детальные сцены взяток, угроз
- Насилие: описательное, кровь через текст
- Эротика: только текстовые намёки

### Визуальные события
- Пиксельные комикс-кадры: силуэты в откровенных позах (пикселизовано)
- «Кино-нуар» подача: тени, свет, недосказанность
- Стиль Papers Please: мрачная документальная эстетика

### Примеры событий в этой версии
```
💊 «КОКАИН В ТУАЛЕТЕ»
«Охрана зашла в мужской туалет и застала троих
с порошком на раковине. Дилер — постоянный клиент.»
[Игнорировать — Police Risk] 
[Выгнать всех — Reputation -5]
[Предложить «крышевание» — Stash +$500]

💋 «VIP-КОМНАТА»
«Ваш охранник Виктор доложил: две знаменитости
заперлись в VIP. Журналист Дэйли Пост ждёт снаружи.»
[Замять — $300] 
[Слить в прессу — Hype +40%, Police Risk]
[Охранять дверь — ничего]
```

## ВЕРСИЯ 3: «Максимум» (Director's Cut / NSFW)

**Возрастной рейтинг:** 18+ (Adult Only на некоторых платформах)  
**Платформы:** Steam (Adult Only DLC), Telegram каналы 18+ (отдельный контент)

### Правила контента
- Наркотики: полные сцены потребления (пиксельная графика)
- Секс: откровенные пиксельные иллюстрации (не хентай — реалистичные 70-е)
- Коррупция: детальные механики, дополнительные NPC
- Насилие: более детальное
- Атмосфера: «грязная», как в реальном basement Studio 54

### Визуальные события
- Полноценные пиксельные комикс-сцены (6–8 кадров)
- Анимированные CG-иллюстрации для ключевых моментов
- Персонажи в стиле «pornochic 70s»: реалистично, но пиксельно
- Basement-уровень: отдельная локация с уникальными механиками

### Дополнительные механики только в Версии 3
- **Basement management:** управляй тем, что происходит в подвале
- **Dealer relationship:** коррупционные связи с дилерами дают уникальные бонусы
- **Blackmail mechanic:** компромат на знаменитостей
- **Vice income:** секс-работники в клубе дают отдельный поток дохода

### Telegram-канал (18+)
Для Telegram отдельный **закрытый канал 18+** с:
- Unlock-кодами для максимальных сцен
- Эксклюзивными иллюстрациями персонажей
- Дополнительными комикс-кадрами
- Реализуется через Telegram Stars или подписку

## ТЕХНИЧЕСКИЕ РЕШЕНИЯ

### Архитектура
```javascript
// GameState.contentVersion = 'safe' | 'adult' | 'max'

// Каждое событие имеет три слоя:
event.title_safe   // для версии 1
event.title_adult  // для версии 2
event.title_max    // для версии 3

event.body_safe   // текст без деталей
event.body_adult  // откровенный текст
event.body_max    // + ссылка на иллюстрацию

event.image_adult // null или путь к пиксельной сцене
event.image_max   // null или путь к полной иллюстрации
```

### Версия 3 — отдельный DLC
```
steam_build_base/    ← Версии 1 и 2
steam_build_dlc_18/  ← Дополнение Версия 3 (отдельная покупка)
```

### Парентальный контроль
- Версия 1 по умолчанию для всех
- Версии 2 и 3 требуют подтверждения возраста
- В Steam: стандартная система Adult Content
- В Telegram: проверка через @BotFather age_verification API
