## АРТИСТЫ — БАЗА ДАННЫХ

### КАК АРТИСТЫ СТАНОВЯТСЯ ДОСТУПНЫ

Репутация клуба → открывает агентов → агенты предлагают артистов.

```
Репутация 0–30:   Местные акты (клубные диджеи, начинающие группы)
Репутация 31–60:  Городские звёзды (региональные имена)
Репутация 61–80:  Национальные артисты
Репутация 81–100: Легенды (Donna Summer, Earth Wind & Fire, etc.)
```

Также влияет Sound System уровень: ур.1 = базовые акты, ур.2 = средние, ур.3 = топовые.

## МЕХАНИКА КОНЦЕРТНОЙ НОЧИ

### Отличия от обычной ночи

| Параметр | Обычная ночь | Концерт |
|---|---|---|
| Длительность | 55 сек | 90 сек |
| Входной поток гостей | 10/мин | 25/мин (×2.5) |
| Тип гостей | Смешанный | Смещён к фанатам артиста |
| Базовый доход | $1,500–3,000 | $5,000–80,000 |
| Риск инцидентов | Обычный | ×2 (больше народу) |
| FBI Suspicion рост | Обычный | +20% за вечер (пресса) |
| Обязательные правила | Обычные | Специальные (см. ниже) |

### Специальные правила концерта
Перед концертной ночью: от артиста приходит «райдер» — список требований.

**Примеры райдеров:**
- *«Donna Summer требует: только безалкогольные напитки за кулисами, никаких фото»*
- *«Earth Fire Wind требует: охрана у каждого выхода, нет папарацци»*
- *«Westside T требует: свой DJ в разогреве, VIP-зона для 20 чел., $5000 наличными»*

Не выполнить требование → артист уходит до начала → ты теряешь аванс.

### Концертный фейсконтроль
Особый режим — гости приходят группами (фанаты знают дресс-код артиста):
- Фанаты Donna Summer — в белом, 70-е стандарт
- Фанаты Run DMZ — в спортивном, нестандарт (это новинка, 80-е)
- Решать, впускать ли «не тех» фанатов — риск конфликта культур

### Логика бронирования
1. Нажал «забронировать» → плата аванса (50% гонорара)
2. Следующие 2 ночи: обычные (время на подготовку)
3. На третью ночь: автоматически запускается концертный режим
4. Если клуб закрылся / рейд в эти 2 ночи → аванс потерян, концерт отменён

### Стоимость подготовки (помимо гонорара)

| Что нужно | Стоимость |
|---|---|
| Временная сцена | $800 |
| Доп. охрана на вечер | $400 |
| Звуковой инженер | $300 |
| PR / реклама концерта | $200 (репутация +10 после) |
| **Итого минимум** | **$1,700 + 50% аванса** |

## РЕПУТАЦИЯ КАК РЕСУРС ДЛЯ КОНЦЕРТОВ

### Почему репутация важна

Крупные артисты не едут в клуб который «неизвестен». Репутация — это гарантия что:
- Пресса придёт
- Фанаты заполнят зал
- Артист не потеряет лицо

### Как поднимать репутацию под концерты

| Действие | Репутация |
|---|---|
| Световое шоу апгрейд ур.2 | +5/ночь |
| Пройти ночь без скандалов | +3 |
| Впустить знаменитость | +5–15 |
| Провести успешный концерт | +15–30 |
| Статья в прессе (положительная) | +20 |
| Скандал / рейд полиции | -15–30 |
| Провалить концерт (мало народу) | -10 |

### Репутационный ущерб от провала концерта
Провал — это когда заполняемость < 60%. Тогда:
- Артист платит штраф за «некачественную аудиторию»
- Пресса пишет «клуб угасает»
- Следующий концерт дороже (агент требует «страховой депозит»)

## НАРРАТИВНЫЕ СОБЫТИЯ КОНЦЕРТОВ

### Эпизоды связанные с концертами

**«Donna Summer — полночный монолог»** *(если репутация 65+)*  
> *«В 2:15 она остановила музыку. Сказала что-то в микрофон. Я не слышал — стоял у входа. Но зал затих. Потом она запела "I Feel Love" без всего — только голос. Тишина. Потом взрыв»*

**«Фанаты снаружи»** *(автоматически при концерте)*  
> Полиция звонит: «У вас снаружи 300 человек. Либо вы их впускаете, либо мы»  
> Выбор: открыть ещё один вход (доход ×1.5, риск давки) / оставить как есть / вызвать охрану

**«Артист опаздывает»** *(15% шанс на любом концерте)*  
> Задержка 30 мин. Толпа начинает нервничать.  
> Решение: DJ играет разогрев (нормально) / выйти объявить самому (хайп или провал) / тихо возмещение

**«Таинственный гость»** *(10% шанс если репутация 80+)*  
> В очереди на концерт стоит кто-то особенный. Выглядит как обычный гость — но это агент лейбла / режиссёр / будущий бизнес-партнёр.  
> Если впустишь и не облажаешься → деловое предложение на следующий день

## TELEGRAM-СПЕЦИФИКА КОНЦЕРТОВ

### Концерт как «Weekly Event»
- Каждую неделю — анонс концерта в официальном канале
- Игроки соревнуются: кто соберёт больше на одном концертном вечере
- Лидерборд «Концерт недели»

### Концертный токен
- Специальный ресурс «Concert Pass» (один в неделю бесплатно)
- Второй Concert Pass — 50 ⭐ Telegram Stars
- Не использовал в эту неделю — сгорает (FOMO)
